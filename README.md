## 📚 About This Project

This repository contains all **Angular projects and files** developed as part of my academic coursework for the year **2024**. These projects were completed under the curriculum of **MSc in Software Technology** at **St Aloysius (Deemed to be University), AIMIT, Mangalore**.

These Angular exercises include beginner to intermediate-level web applications, covering key concepts like:
- Components and Modules
- Services and Dependency Injection
- Event and Property Binding
- Form Handling
- CRUD operations
- Routing and Navigation
